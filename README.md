Release Notes Rake Task
====================================================

A quick and dirty rake task that produces release notes from git
history:

```
Previous Tag: v0.1.1

@npverni
[bugfix] Fixes bug in UI

@githubusername
[feature] Adds support for XYZ
```

### Installation

Simply copy this file into your `lib/tasks` directory.


### Usage

```bash
$ rake release_notes`
```
